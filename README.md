#### 课程作业要求
期末大作业用java实现一个计算器，要求实现以下功能：
1. 加减乘除，三角函数 30分
2. 幂，求导、阶乘 30分
3. 画图:统计图(柱状、饼状等)、回归曲线(输入一组数据(s,y)，预测y曲线并画出)、三角函数图/求导/其他 30分
4. 创意功能 10分
